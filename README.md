# 🍕 Pizza Sales Dashboard

### Microsoft Excel | SQL | Data Analytics

An interactive **Pizza Sales Dashboard** built in **Microsoft Excel** to analyze sales performance, customer purchasing behavior, and business trends through dynamic visualizations and key performance indicators (KPIs).

---

## 📌 Project Overview

The **Pizza Sales Dashboard** is an interactive business intelligence dashboard developed in **Microsoft Excel** to analyze pizza sales data and uncover actionable business insights. It enables users to monitor key performance indicators (KPIs), identify sales trends, evaluate product performance, and understand customer purchasing behavior through dynamic visualizations.

The dashboard was built using **Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and SQL** to transform raw sales data into an easy-to-understand and interactive reporting solution.
