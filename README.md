# 🍕 Pizza Sales Dashboard | Microsoft Excel

## 📌 Project Overview

This project is an interactive **Pizza Sales Dashboard** built in **Microsoft Excel** to analyze pizza sales performance and generate meaningful business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, analyze customer purchasing behavior, and evaluate product performance through interactive visualizations.

The project combines SQL for KPI calculations and Excel for dashboard creation using Pivot Tables, Pivot Charts, Slicers, and data visualization techniques.
